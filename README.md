#  Resume RAG Chatbot (Local LLM + LangChain)

A fully offline **Retrieval-Augmented Generation (RAG)** chatbot that answers questions from my resume and documents using embeddings + vector search + a local LLM.

This project demonstrates **real AI Engineering skills**:
- LLM integration
- Vector databases
- RAG pipelines
- Backend APIs
- Local GPU inference
- Production-style architecture
